# Python OOP 30+ Examples

This folder contains 36 beginner-to-practical OOP examples.

## Naming Format
Each file follows this format:

```text
1_<topic>.py
2_<topic>.py
...
```

## How to Run

```bash
python 1_procedural_vs_oop.py
python 2_class_and_object.py
```

For BeautifulSoup and PyMySQL examples, install requirements:

```bash
pip install -r requirements.txt
```

## Notes

- Examples 1 to 32 cover core OOP concepts.
- Example 33 covers SQLite database with OOP.
- Example 34 covers PyMySQL database with OOP.
- Examples 35 and 36 cover BeautifulSoup scraping using OOP.
