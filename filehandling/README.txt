Compression Examples - ZIP, GZIP, TAR, TAR.GZ

Files included:
1. students.txt
2. sales.csv
3. config.json
4. 01_zip_example.py
5. 02_gzip_example.py
6. 03_tar_example.py
7. 04_targz_example.py

How to run:
Open terminal inside this folder and run:

python 01_zip_example.py
python 02_gzip_example.py
python 03_tar_example.py
python 04_targz_example.py

These scripts will generate:
- sample_files.zip
- students.txt.gz
- sample_files.tar
- sample_files.tar.gz
- extracted output folders
